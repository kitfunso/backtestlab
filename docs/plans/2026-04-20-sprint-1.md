# Sprint 1 — MCX commodities + extended portfolio metrics

**Goal:** Add 6 MCX commodities as first-class instruments in the strategy tester, and surface portfolio-level metrics (rolling Sharpe/vol, top-5 drawdowns, return distribution, Sortino, Calmar, diversification ratio) in the Portfolio panel.

**Prerequisites:**
- Backtest Lab repo scaffolded and live at `backtestlab.pages.dev`
- `docs/PRD.md`, `docs/ARCHITECTURE.md`, `CLAUDE.md` approved
- MCX commodity list confirmed (from CEO's Excel): GOLD, SILVER, COPPER, CRUDE, MCXBULLDEX, MCXMETLDEX

**Estimated scope:** 15 steps, ~1 week human / ~1–2 working sessions with CC

### Step 2: Write MCX bhavcopy ingestion script

**Files:**
- `scripts/ingest_mcx.py` (new)
- `scripts/requirements.txt` (new — `requests`, `pandas`, `python-dateutil`)

**What:** Python script that:
1. Fetches today's MCX bhavcopy CSV from `https://www.mcxindia.com/market-data/bhavcopy`
2. Filters to the 6 registry symbols
3. For each symbol, appends the day's row (date, open, high, low, close, volume) to `public/india/prices/mcx/{SYMBOL}.json`
4. Creates the file if it doesn't exist; no duplicates on re-run

**Verify:**
```bash
python scripts/ingest_mcx.py --dry-run
```
Expect: logs listing 6 symbols with today's prices; no files written.

```bash
python scripts/ingest_mcx.py
ls public/india/prices/mcx/ | wc -l    # expect: 6
```

**Commit:** `feat(mcx): add daily bhavcopy ingestion script`

### Step 3: Historical backfill from MCX archives

**Files:**
- `scripts/backfill_mcx.py` (new)
- `public/india/prices/mcx/*.json` (populated)

**What:** Downloads 12 months of MCX daily bhavcopy zips, extracts, consolidates per-symbol OHLCV. MCX publishes these at `https://www.mcxindia.com/market-data/bhavcopy?dDate={DD/MM/YYYY}`. For instruments that predate MCX index launches (MCXBULLDEX ~2020), start from whenever the series begins.

**Verify:**
```bash
python -c "import json; d=json.load(open('public/india/prices/mcx/GOLD.json')); print(len(d['dates']), d['dates'][0], d['dates'][-1])"
```
Expect: ~250+ trading days of history, spanning 12+ months.

**Commit:** `data(mcx): backfill 12 months of historical prices for 6 commodities`

## Part C — Portfolio metrics module

### Step 8: Rolling Sharpe / rolling vol

**Files:**
- `src/lib/portfolio/metrics.ts` (new)
- `src/lib/portfolio/metrics.test.ts` (new — unit tests)

**What:** Functions:
- `rollingVolatility(returns: number[], window: number = 126): number[]` — 6-month (126 trading days) rolling annualized vol
- `rollingSharpe(returns: number[], window: number = 126, rfRate: number = 0.06): number[]` — 6-month rolling annualized Sharpe, risk-free default = 6% INR
- Unit tests with known-answer cases (e.g. constant returns → vol = 0; positive mean → positive Sharpe)

**Verify:**
