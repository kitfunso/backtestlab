**Goal:** Turn Backtest Lab from a hand-picked strategy tester into a goal-driven portfolio builder. Ship (a) a price-derived multi-factor library (momentum / low-vol / low-beta / reversal) feeding a goal-based wizard that auto-picks 12 stocks, (b) a 10-min RSS news feed tagged per stock, (c) localStorage-backed saved portfolios.

- Sprint 1 shipped at v0.2.1 (MCX commodities + real bhavcopy + TC model + portfolio metrics)
- 205 NSE price JSONs in `public/india/prices/` and 6 MCX JSONs in `public/india/prices/mcx/`
- CEO confirmed scope: multi-factor (all typical equity factors), 204 universe, 10-min RSS, Resend email, keep localStorage v1

**Out of scope (pushed to Sprint 3):**

- **Rebalance email alerts** — requires Supabase auth + user accounts + cron worker; bundled with account system in Sprint 3
- **Cross-device sync** — same dependency as alerts (Supabase)
- **DCA vs lump-sum simulator** — deferred to Sprint 2.5 if time; otherwise Sprint 3

## Part A — Price-derived factor library

### Step 1: Factor definitions module

**Files:**
- `src/lib/factors/definitions.ts` (new)
- `src/lib/factors/definitions.test.ts` (new)

**What:** Pure functions operating on `number[]` (daily close series) returning a single factor score. Definitions:
- `momentum12_1(closes)` — return from t-252 to t-21 (skips recent month)
- `momentum6_1(closes)` — return from t-126 to t-21
- `shortReversal(closes)` — negative of last 5-day return
- `volatility(closes)` — 252-day annualized vol (lower = better factor score)
- `downsideBeta(closes, marketCloses)` — beta conditional on market down days
- `dispersion(closes)` — 21-day intraday range / price average

All return `number | null` (null when insufficient history).

**Verify:**
```bash
npx jest src/lib/factors/definitions.test.ts
```
Expect: known-answer cases pass (constant series → momentum=0, vol=0; monotonic up → positive momentum).

**Commit:** `feat(factors): price-derived factor definitions (momentum, vol, beta, reversal)`

### Step 2: Cross-sectional rank + z-score utility

**Files:**
- `src/lib/factors/rank.ts` (new)
- `src/lib/factors/rank.test.ts` (new)

**What:** `crossSectionalZScore(values: Record<ticker, number | null>): Record<ticker, number>`. Tickers with null get z=0 (neutral). Winsorize at ±3σ before z-score (robustness).

**Verify:**

Expect: uniform distribution gives z near [-1.7, 1.7]; outlier gets winsorized to ±3.
